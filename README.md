# upifarm landing page

Premium static landing page for upifarm, a seasonal Vietnamese produce brand.

## Direction

- Brand-first landing page, not ecommerce.
- Focused on trust, seasonal selection, and direct contact.
- Products: Musang King, Bơ 034, Măng cụt, Cà phê rang mộc.
- Built with plain HTML, CSS, and JavaScript for Cloudflare Pages.

## Deploy on Cloudflare Pages

- Framework preset: None
- Build command: leave empty
- Output directory: `/`
